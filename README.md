# The Holy Qur'an Web App

A modern, fast, and responsive Qur'an web application built with **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**, featuring SSR search, dynamic ayah rendering, and user customizable reading settings.

---

## Features

- 114 Surahs listing with Arabic & English names
- SSR-based search for Ayahs (server-side fetching)
- Detailed Surah page with all verses
- User settings (font, font size, translation size)
- Persistent preferences using LocalStorage
- Smooth suggestion dropdown for search
- Fully responsive UI (mobile-first)
- Optimized SSR + client components separation

---

## Tech Stack

- **Next.js (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **Redux Toolkit**
- **LocalStorage (custom persistence)**
- **Server Actions (SSR fetching pattern)**
